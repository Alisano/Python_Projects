Программа для расчета распределения полей, потенциалов, средней энергии электронов в пироЭОП\
Инструкция по запуску программы:
1. Перейдите в рабочую папку
2. Скомилируйте array_figures_min_step.cpp в исполняемый фаил
3. В файле imager_model.py в переменной cmd допишите полный путь до скомпилированного фаила, включая название фаила
3. Не забудьте в конце строки cmd дописать " {cmd_param}", включая пробел. Данный параметр нужен для передачи параметров из текущего python скрипта в исполняемый скомпилированный фаил
4. Удостоверьтесь что скомпилрованный код и python скрипт лежат в одной рабочей папке. Запустите в рабочей папке python скрипт. После расчета (время расчета может быть продолжительным), в рабочей папке появятся два текстовых файла с данными расчета, по которым скрипт построит графики.
